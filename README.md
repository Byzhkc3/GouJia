# GouJia
  用ionic1开发的构家网app

  项目地址:  [Byzhkc3/GouJia](https://github.com/Byzhkc3/GouJia)

### 安装

``` bash
# 安装依赖
cnpm install / npm install

# 运行
ionic serve

# 打包
ionic platform add ios / ionic platform add android
ionic build ios / ionic build android

```
### 下载安装包
